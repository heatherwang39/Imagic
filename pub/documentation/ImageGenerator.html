addElements(eventType, totalSelectoropt, seperateSelectorsopt, elementCodeopt, elementColoropt, elementSizeopt)

Add multiple repeating elements as the response to user's operations to each image in this group. The position of the added elements is the position of the user's mouse. The added elements can be removed by clicking on itself again.

Parameters:
Name Type Attributes Description
eventType string

The way for user to interact, for example, "click", "mouseover" or "keydown".
totalSelector string <optional>

(optional)The css selector of where the developer wants to update the information of the total counter for all images in this group. Set as "null" by default when the developer doesn't need to show the counter in the website. Can be any CSS selectors, for example, "#totalCounter".
seperateSelectors Array.<string> <optional>

(optional)The css selector of where the developer wants to update the information of the seperate counter for each image in this group. Set as "null" by default when the developer doesn't need to show the counter in the website. It should be a list of CSS selectors and has the same length as the imageUrls. Can be any CSS selectors, for example, "#seperateCounter1".
elementCode string <optional>

(optional)The html code for the elements,"♥" by default(♥). Can be any html codes, for example, "⚑"(flag-&#9873),"☀"(sun-&#9728).
elementColor string <optional>

(optional)The color of the elements, "red" by default. Can be "random"(will choose random color for each interactions) or any other colors, for example, "blue", "rgb(0,0,0)" or "#123456".
elementSize string <optional>

(optional)The size of the elements, "200%" by default. Can be any size, for example, "100%" or "500%".
